# TMM Simulator

Private binary release page for the TMM Simulator.

This public repository does not contain source code.  
Download the executable package from the Releases page.

## Download

Go to **Releases** and download the file for your OS.

- macOS Apple Silicon: `TMM-Simulator-Darwin-arm64-...zip`
- Windows: coming soon

## Notes

The app runs locally in your browser after launching the executable.
Source code and research implementation details are not distributed in this public repository.
